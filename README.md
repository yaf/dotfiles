Installation
============

install with
`mkdir -p ~/.config/dotfiles && git clone git@github.com:yaf/dotfiles.git ~/.config/dotfiles && .config/dotfiles/setup`

Mostly inspired by JulienXX dotvim https://github.com/julienXX/dotvim
